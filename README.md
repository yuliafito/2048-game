# 2048 Game
2048 Game is a classic puzzle game implemented using pure JavaScript. The goal of the game is to combine tiles with the same values on a 4×4 board and reach the 2048 tile.

## Live Demo
Experience the live website: [2048](https://yuliafito.github.io/2048-game/)

## Technologies Used
- **JavaScript** – game logic
- **HTML5** – page structure
- **SCSS** – styling and layout
- **DOM Manipulation** – dynamic UI updates
- **Keyboard Events** – arrow key controls

## Getting Started
Clone the repository:
```bash
git clone https://github.com/yuliafito/2048-game.git
cd 2048-game
```
Install dependencies:
```bash
npm install
```
Run the project locally:
```bash
npm start
```

## Features
- Classic 2048 gameplay
- Keyboard controls
- Score calculation based on merged tiles
- Win message when the 2048 tile is reached
- Game Over message when no moves are available
